# laravel框架下的sl

#### 介绍
php的laravel框架作业

#### 软件架构
laravel框架


#### 安装教程

1. sl里有members.sql 导入进去就行了
2. 修改sl\.env的数据库信息与sl\config\database.php的数据库信息
3. 功能在使用说明那里

#### 使用说明

1. 输入 localhost\sl\public\index 是显示列表
2. 输入 localhost\sl\public\delindex 是被删除显示列表
3. 具体看sl\routes\web.php
